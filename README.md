## Tecnologias

O projeto foi desenvolvido com as seguintes Tecnologias:

- React
- Typescript
- Storybook
- Testes (unitários do com Jest)
- MSW
- Addons de usabiidade e testes para Storybook
- Radix UI
- Tailwind CSS
- CI/CD
- Github Pages
